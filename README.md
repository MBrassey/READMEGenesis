## READMEGenesis :scroll:

Menu driven command line utility written in Node.js to generate a professional README for new projects.

Usage: node index.js [ -h | -v | -l | -a ]

[options]

    -v          Show version.
    -l          Show license info.
    -a          What is READMEGenesis?
